# dockerstats package

## Usage
### Repo
Clone this repo, run pip install -r requirements.txt <br/>
then run python main container-a container-b

### Pip packaging
Create a package shown here: https://github.com/Kreidl/own-pip-package<br/>
or install it from: https://pypi.org/project/dockerstats/<br/>
Then you can run it with main container-a container-b
